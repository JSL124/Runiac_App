# Runiac Test Evidence Instructions

## Scope
- Applies to QA screenshots, manual test records, emulator output, and verification reports under `test-evidence/`.
- A12_QA_TEST owns evidence capture and reporting.
- This folder must not contain production source code or automated test definitions.

## Evidence Rules
- Record what was tested, what passed, what failed, and what was not run.
- Avoid precise private GPS routes, health/safety readiness details, personal profile data, and secrets in screenshots or logs.
- Keep evidence tied to PRD/PDD traceability when implementation begins.
