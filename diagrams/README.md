# Runiac Diagrams

This folder stores diagram planning notes and source files for the Runiac PDD.

## Structure

| Folder | Purpose |
| --- | --- |
| `physical_architecture/` | Physical Architecture Diagram assets, Mermaid source, and explanation notes. |
| `application_architecture/` | Application Architecture Diagram Mermaid source and explanation notes. |
| `class_diagram/` | Final UML Class Diagram reference, simplified PDD Class Diagram drafts, detailed reference Class Diagram sources, exports, and planning notes based on Topic 3 UML rules. |
| `component_diagram/` | Component Diagram source, exports, and planning notes based on Topic 4 UML rules and Runiac architecture references. |

## Current Status

| Diagram | Status |
| --- | --- |
| Physical Architecture Diagram | Current draft captured from the user-provided screenshot and saved as Mermaid/source notes. |
| Application Architecture Diagram | Initial Mermaid draft created from PRD features and wireframe module structure. |
| Class Diagram | Final screenshot reference saved, with structured text reference plus earlier Draw.io/SVG/PNG exports. |
| Component Diagram | Draw.io source plus SVG/PNG exports generated for the PDD. |
